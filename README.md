# Telegram-бот «Виселица»

Этот проект реализует игру «Виселица» в виде Telegram бота с помощью библиотеки telebot (pyTelegramBotAPI). 
Ссылка на бот: t.me/valeri_hangman_bot
** *Примечание:* ** *бот может быть не доступен в определенный момент, так как используется бесплатный тариф на платформе PythonAnywhere, попробуйте запустить его снова немного позднее.*

## Запуск 

1. Установите зависимости:
   ```bash
   pip install pyTelegramBotAPI
   ```
2. Определите переменные окружения:
   ```bash
   export YOUR_BOT_TOKEN="123:ABC"
   ```
3. Запустите бота:
   ```bash
   python the_gallows_game.py
   ```

## Возможности
- Генерация случайного слова из словаря и отображение текущего состояния виселицы.
- Проверка введённых букв и слов с отслеживанием уже использованных попыток.
- Обработка победы и поражения с предложением сыграть заново.
- Поддержка одновременных игр в разных чатах.

## Данные 
Словарь для игры взят из следующего репозитория:
https://github.com/Harrix/Russian-Nouns/blob/main/dist/russian_nouns.txt

## Команды
- `/start`, `/help` - краткая справка по игре и списку команд.
- `/new` - запуск новой игры или перезапуск после окончания.
- `/stop` - досрочное завершение текущей игры.

После победы или поражения бот задаёт вопрос «Хотите сыграть ещё раз?», ожидая ответ `да` или `нет`, чтобы продолжить игру или ее завершить.
